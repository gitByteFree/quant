# 因子研究报告

## 1. 因子检验框架

### 1.1 IC分析（信息系数）

对每个交易日截面计算因子值与未来收益的相关系数：

$$IC_t = \text{corr}(FactorValue_t, ForwardReturn_{t+h})$$

**评估标准**：

| 指标 | 计算公式 | 合格线 |
|---|---|---|
| IC均值 | $\bar{IC} = \frac{1}{T}\sum IC_t$ | \|IC\|>0.02 |
| IC标准差 | $\sigma_{IC}$ | - |
| IC_IR | $IC\_IR = \frac{\bar{IC}}{\sigma_{IC}}$ | >0.3 |
| IC>0比例 | $\frac{\sum \mathbb{1}(IC_t > 0)}{T}$ | >55% |
| t统计量 | $\frac{\bar{IC}}{\sigma_{IC}/\sqrt{T}}$ | \|t\|>2.0 |

### 1.2 Fama-MacBeth回归

两步法估计风险溢价：

**第一步**（截面回归，每期）：
$$R_{i,t} = \alpha_t + \sum_{j=1}^k \beta_{j,t} \cdot Factor_{i,j,t-1} + \epsilon_{i,t}$$

**第二步**（时间序列平均）：
$$\hat{\lambda}_j = \frac{1}{T}\sum_{t=1}^T \hat{\beta}_{j,t}$$

对 $\hat{\lambda}_j$ 做t检验判断因子显著性。

### 1.3 分层回测

每个交易日按因子值将股票分为10组，等权持有，计算每组累计收益。

- Top-Bottom收益差应显著>0
- 各组收益应呈现单调性

### 1.4 因子衰减分析

计算不同预测周期上的IC衰减：

$$IC(h) = \text{corr}(FactorValue_t, \sum_{\tau=1}^h R_{t+\tau})$$

**半衰期**: IC均值衰减到最大值一半的周期数。半衰期越长，因子信号越持久。

## 3. 因子预处理效果

### 3.1 去极值（MAD法）

$$Factor_{clipped} = \text{clip}(Factor, median \pm 3 \times MAD)$$

效果：
- 消除极端值对IC的影响
- 提高因子稳定性
- 不会损失因子区分度

### 3.2 中性化

$$Factor_{neutral} = Factor - \hat{Factor}_{ind+size}$$

效果：
- 消除行业偏好（不在某些行业集中）
- 消除市值偏好（不纯粹是大小盘轮动）
- 提高因子纯度

### 3.3 标准化

$$Z\_Score = \frac{Factor - \mu_{cross}}{\sigma_{cross}}$$

效果：
- 截面可比性
- 便于多因子组合
- 提高模型训练稳定性

## 4. 因子相关性矩阵（预期）

动量类和波动类因子通常负相关；价值类和质量类正相关；情绪类独立。

建议选IC_IR>0.3且两两相关性<0.7的因子进入模型。
